# Code of Conduct — Conductra Health

## Our Commitment

Conductra Health is committed to providing a welcoming and professional environment for all contributors, collaborators, and community members.

We are building software for hospice care — an environment that demands compassion, precision, and respect. We hold ourselves to the same standard in how we operate as an organization.

---

## Expected Behavior

- Communicate professionally and respectfully
- Focus feedback on code, architecture, and process — not individuals
- Respect the mission: this software supports end-of-life care
- Protect patient privacy as an absolute priority
- Follow security and IP guidelines in every repository

---

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks of any kind
- Sharing or exposing Protected Health Information (PHI)
- Committing proprietary IP (weights, formulas, signal inputs) to public repositories
- Circumventing security controls or access restrictions

---

## Scope

This Code of Conduct applies to all Conductra Health GitHub repositories, issues, pull requests, and any other project spaces.

---

## Enforcement

Violations may be reported to: security@conductrahealth.com

All reports will be reviewed by Kevin Grant (Conductra Health founder). Appropriate action will be taken, up to and including removal from the project.
